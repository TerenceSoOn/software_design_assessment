**FlirtNet -- An Advanced Online Dating Platform**


# Base features
1. Random matching to chat. Users can see the profile the user that are also using this function, to choose OK/NotOK. If user A choose OK on user B and so does user B choose OK on user A, they will be matched and start a private chat.
2. Having a place where users can publish their posts (text&images), others can like, comment, or start a private chat directly if both user agree.
3. Personal profile page, photo wall
4. Private chat & Group chat


# AI featurs
1. If you cannot match anyone for long time, AI will pretend itself as a real user to spend some time with you as compensation.
2. AI can pretend to be your lost partner. If you talk with someone for a long time and you finally lose him/her, you can ask AI to imitate his/her tone to continue to chat with you.
3. AI can also help with matching users.
4. AI Wingman: Based on your chatmate's profile, AI will provide you some advices on how to start the chat.
5. AI can help you distinguish what the atmosphere between you two is like currently
6. AI profile optimizor
7. Practice with AI before actually chat with someone
