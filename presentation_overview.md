## Part 4: Architecture & Design


**1. Design Pattern**

- **MVC:**
  - Model: Database Model
  - View: React (Frontend UI)
  - Controller: FastAPI (Business logic)

**2. Tech Stack**

- Frontend: React
- Backend: FastAPI
- Database: SQLite
- Real-time: Socket.IO

**3. UI Design**

- Liquid Glass (iOS-inspired)
- Glassmorphism: Translucent cards, blur effects
- Pink/Purple gradients since it's a dating platform
